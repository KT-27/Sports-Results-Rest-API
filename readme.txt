Installation guide:

Extract the repository in a folder of your choice.
Open a console /// and initialise virtual environment
Install the requirements:
'pip install -r requirements.txt'

Start the server:
'python manage.py runserver'
Access the server at 127.0.0.1:8000

The swagger schema provides detailed documentation about the available API endpoints.
